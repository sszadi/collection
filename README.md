# Insole web application

Insole is a place where you can create your own sneakers collection and make it bigger.

### Modules:
-----------------------------------------------------------------------------------------------------------------------

  - Registration and login with Spring Security
  - Sneakers collection
    - CRUD operations
    - Uploading images to AWS S3
    - Wish list
    
### Technologies:
-----------------------------------------------------------------------------------------------------------------------

- Spring Data
- Spring MVC
- Spring Security
- JSP
- JSTL, Thymeleaf
- Bootstrap
- jQuery
